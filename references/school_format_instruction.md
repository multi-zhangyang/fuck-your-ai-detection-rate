# 学校论文格式说明
一、论文封面
封面中中文题目为小2黑体加粗居中，英文题目为小2号Times New Roman加粗居中，其他所填内容为小3号宋体居中。
二、目录
目录标题：3号黑体居中，两个字中间空两格，上面空一行，段前0行、段后1.5行，1.5倍行距。目录内容用小4号宋体，1.5倍行距。
三、中英文摘要、关键词
摘要：3号黑体居中，两个字中间空两格，上面空一行，1.5倍行距；内容用5号宋体，行间距固定值20磅。
关键词：5号黑体加粗；内容用5号宋体，行间距固定值20磅。
Abstract：3号Times New Roman加粗居中，上面空一行，1.5倍行距；内容用5号Times New Roman，行间距固定值20磅。
Key words：5号Times New Roman加粗；内容用5号Times New Roman。
四、主体部分
1 ××××：一级标题，4号黑体，居左。1.1 ××××：二级标题，小4号黑体，居左。1.1.1 ××××：三级标题，5号黑体，居左。1.1.1.1 ××××：四级标题，5号黑体，居左。
论文正文用5号宋体，行间距固定值20磅，两端对齐。正文所有数字和字母均使用Times New Roman字体。
图序、图名放在插图下方，居中，字体为小5号黑体。图注小5楷体。
表序和表名放在表格上部，居中，字体为小5号黑体。表格内容字体为小5号宋体，数字用小5号Times New Roman。表注小5楷体。
五、参考文献
参考文献：4号黑体居中，上面空一行，另起一页，1.5倍行距；内容文字用5号宋体，数字字母为Times New Roman，行间距固定值20磅。
六、致谢
致谢：3号黑体居中，两个字之间空两格，上面空一行，另起一页，1.5倍行距；内容用小4号宋体，行间距固定值20磅。
八、页面格式、打印
A4版面，页边距上2.5cm、下2.5cm、左3cm、右3cm。
